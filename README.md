# Análise de Projetos com Machine Learning 

## 📈 Foco: Regressão Linear
O objetivo principal é utilizar **Regressão Linear** para prever o custo estimado dos projetos com base no seu percentual de execução, servindo como ferramenta de auditoria e planejamento.

## 🛠️ Tecnologias
- Python (Pandas, Scikit-learn, Folium)
- Algoritmos: K-Means e Regressão Linear

## 📂 Como visualizar
- Abra o arquivo `analise_ml.ipynb` para ver o código e os gráficos.
- Abra o arquivo `mapa_projetos_ppi.html` para ver o mapa interativo.
